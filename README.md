# Веб-сайт школы с бэкендом на Go

Это проект современного сайта-визитки для образовательного учреждения, включающий в себя фронтенд (HTML/CSS/JS) и бэкенд на Go для обработки контактной формы.

## Функциональность

- **Фронтенд:** Адаптивный одностраничный сайт с различными секциями и интерактивными элементами.
- **Бэкенд:** Веб-сервер на Go, который:
    - Обслуживает статический сайт (`index.html`).
    - Предоставляет API-эндпоинт (`/api/contact`) для приема сообщений из контактной формы.
    - Сохраняет полученные сообщения в локальную базу данных SQLite (`school.db`).

## Требования

- **Go**: Установленная и настроенная среда Go (версия 1.18 или выше).

## Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <адрес-репозитория>
    cd <папка-репозитория>
    ```

2.  **Установите зависимости:**
    Выполните команду в корневой папке проекта, чтобы загрузить необходимые Go-модули (в данном случае, драйвер для SQLite).
    ```bash
    go mod tidy
    ```

## Запуск приложения

Для запуска веб-сервера выполните следующую команду:

```bash
go run .
```

Сервер будет запущен и доступен по адресу `http://localhost:8080`. Вы можете открыть этот адрес в вашем браузере.

## Как это работает

1.  При запуске `go run .` стартует веб-сервер.
2.  Сервер инициализирует базу данных `school.db` и создает таблицу `contacts`, если они не существуют.
3.  При открытии `http://localhost:8080` в браузере загружается сайт.
4.  Когда пользователь заполняет и отправляет контактную форму, фронтенд отправляет `POST` запрос с данными на `/api/contact`.
5.  Бэкенд на Go получает запрос, сохраняет данные в базу данных `school.db` и отправляет обратно ответ об успехе.
6.  Все полученные сообщения можно найти в файле `school.db`, который является файлом базы данных SQLite.
